# vim常用命令记录

|   快捷键    |              说明               |
| :---------: | :-----------------------------: |
|     dd      |           剪切当前行            |
|      p      |              粘贴               |
|     yy      |           复制当前行            |
|      x      |          删除当前字符           |
|   shift+G   |           到文件顶部            |
|     gg      |           到文件底部            |
|     dw      |         按单词跨度删除          |
| :set number |            显示行号             |
| /[keyword]  | 搜索，按n下一个，shift+n 上一个 |
|   shift+v   |          进入视图模式           |
